LightRail MVC

LightRail is a light weight MVC for jQuery development.

v.0.1

Requirements

  • Requires jQuery

API

				// new instance of lightModel
				var myLightModel = $.extend(lightModel);
			
				// set property
				myLightModel.set({'prop':'val'});
				
				// get property
				myLightModel.set({'prop':'val'});
				
				// on
				myLightModel.on('change',function(){ console.log( this ); });

				// trigger
				myLightModel.trigger('change');
